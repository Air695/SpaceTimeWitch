## 简介

新增可玩角色 时空魔女（SpaceTimeWitch） --通过本体卡牌、额外资源、额外牌堆和扩展体系来进行游戏。
目前仅支持正式版。

## 内容

 - 每局游戏随机获取的扩展体系
 - 通过本地卡牌来在每场战斗中灵活获取扩展体系的卡牌用于战斗。
 - 扩展体系随着游戏进行而成长，大部分使用每幕迭代，小部分依赖局外成长。
 - 扩展体系一般来自于其他我所接触过的作品，现有四个扩展体系。
 - 每个扩展体系一般提供6张基本卡牌、一张多人游戏卡牌。

### 额外牌堆和额外资源
 - **个人空间** 每场战斗中生效的额外牌堆，可以通过效果来对其进行存取/消耗等操作，也参与一些体系。

### 额外资源 
 - **时痕** 用于部分卡牌消耗。

### 本体卡牌
74张可获取卡牌 + 2张衍生卡 + 1张诅咒
### 扩展卡牌
40张卡牌和其升级版本及衍生卡共110张

### 本地化
中文本地化支持和Ai翻译的英文本地化

## 现有扩展体系
不计入升级变体。

- ### 废墟图书馆-流血
施加debuff并触发其效果，在战斗内提升等级并获取额外能力以及额外卡牌。

以施加流血和拼点作为核心手段。

|        |                                                           |
|--------|-----------------------------------------------------------|
| 等级     | 通过战斗获取情感，在正负情感相加达到满值后右键遗物以提升情感等级，每幕有不同情感等级上限。             |
| 升级     | 根据情感值和生命值比例来抽取异想体能力，并在4级和5级时抽取E.G.O卡牌。                    |
| 流血     | 通过卡牌施加流血debuff，敌人会在攻击后受到等同层数的伤害并降低层数为2/3，部分卡牌可从流血中获取额外收益。 
| 拼点     | 通过卡牌和敌人攻击的数值比拼来获取情感、追加伤害并触发额外效果。                          |
| 其他内容 | 6+1张卡牌和5张E.G.O和15个异想体。                                    |

- ### 我的世界-基本
在战斗结束时获取材料并在篝火用以提升卡牌。

以材料获取和局外成长作为核心手段。

|       ||
|-------|-|
| 材料获取  | 可升级的获取材料的遗物。      |
| 局外成长  |在篝火处替换和升级当前携带的卡牌。| 
| 其他内容  |6+1个卡槽共计19张卡牌+1个遗物。|

- ### 艾尔登法环-战技
将不同卡的效果和行动进行交换。

使用核心卡来将下一个战技的行动替换为上一个战技的行动。

5+1张战技及其各自的不同行动，以及一张核心卡。

目前核心卡需要额外通过卡牌获取，体验并不流畅。

- ### 鬼泣-幻影剑
获取、提升并释放大量幻影剑。

使用卡牌获取并使用幻影剑，提升其数值，存入个人空间，最后重复释放。

6+1张卡牌和一张衍生卡。
